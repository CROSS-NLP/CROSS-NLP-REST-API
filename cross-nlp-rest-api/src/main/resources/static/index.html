About : CROSS NLP

Features a vast array of Natural Language Processing tools developed for processing textual data related to the Open Source Software development.

Tools range from pre-processing of text to classifiers and analysers.

  • Code Detector
  • Content Analyser
  • Core Analyser
  • Documentation Analysis
  • Emotion Detector
  • License Detector
  • Plain Text Converter
  • Readability Analyser
  • Reference Detector
  • Request Reply Classifier
  • Senticent Analyser
  • Sentiment Analyser
  • Severity Predictor
